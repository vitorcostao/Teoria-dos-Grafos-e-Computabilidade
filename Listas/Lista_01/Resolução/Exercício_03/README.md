## Exercício 03 (e)

Um grafo simples com sequência de graus (2, 2, 2, 2, 3)

### Resolução

Esse grafo não é possível, pois o somatório dos graus dos vértices resulta em 11, e, pelo teorema do aperto de mão, o número de arestas
seria 5,5 o que é impossível.
