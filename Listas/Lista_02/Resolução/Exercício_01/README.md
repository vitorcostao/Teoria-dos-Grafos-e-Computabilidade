# Teoria dos Grafos e Computabilidade - Lista 02

## Exércicio 01 

Como o número de componentes conexos de um grafo pode mudar se uma nova aresta for adicionada?

### Resolução

Ao adicionar uma aresta, é possível que o número de componentes conexos permaneça o mesmo
ou diminua, pois para aumentar o número de componentes conexos seria necessário aumentar
os vértices.
